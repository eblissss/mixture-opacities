### Fetching Data Instructions
Our data source is https://aphysics2.lanl.gov/

- Have Python Installed and use virtual environment
    - Run `python3 -m venv .venv`
- Then install requirements
    - Run `pip install selenium`
- Download the [ChromeDriver](https://googlechromelabs.github.io/chrome-for-testing/)
    - Add this to PATH
- Run `python fetch_data.py` to execute fetching
